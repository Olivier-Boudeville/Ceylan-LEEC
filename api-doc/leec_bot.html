Description

This module helps the usage of certification bots, in pratice cerbot (see https://certbot.eff.org/pages/about).

The prerequistes and conventions described in https://leec.esperide.org/#wildcard-domain-certificates-with-the-dns-01-challenge are expected to have been respected here.

Function Details

get_certbot_executable_path/0

get_certbot_executable_path() -> executable_path()

Returns a path to the certbot executable.

Note that any plugin for a DNS provider of choice shall have been installed.

For example, on Arch Linux with the OVH DNS provider: pacman -Sy certbot certbot-dns-ovh.

Throws an exception if the executable is not found.
