Overview

The Ceylan-LEEC library (a fork of letsencrypt-erlang) allows to interact from Erlang/OTP with servers implementing the ACME protocol - specifically Let's Encrypt servers, mostly in order to generate X.509 certificates.

This website presents the documentation of the API (Application Programming Interface) of LEEC version 1.2.6 (as of Saturday, April 19, 2025).

Ceylan-LEEC relies on the Myriad layer, a generic Erlang toolbox.
